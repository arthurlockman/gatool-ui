# gatool - Game Announcer Tool

## What is gatool?

gatool provides FIRSTÂ® Game Announcers with up-to-date information while announcing events during the FIRST Robotics season. As a web-based Progressive Web App (PWA), it uses real-time information about events to provide a comprehensive set of useful data to Game Announcers and MCs. It is designed to work on desktops, laptops, and tablet devices. In a pinch, it can be used on a mobile device.

## Key Features

- **Dual Program Support**: Works with both FRC (FIRST Robotics Competition) and FTC (FIRST Tech Challenge)
- **Real-time Data**: Automatically syncs with FIRST APIs and The Blue Alliance
- **Offline Capable**: Progressive Web App that works without internet connectivity
- **Team Management**: Edit and annotate team information for announcements
- **Match Scheduling**: View and track practice, qualification, and playoff matches
- **Live Rankings**: Real-time team rankings with qualification statistics
- **Alliance Selection**: Guided interface for alliance selection process
- **Playoff Brackets**: Visual brackets for 2, 4, and 6 alliance tournaments
- **Awards Management**: Comprehensive award announcement tools
- **Statistics**: Event and world record tracking
- **Play-by-Play**: Real-time match commentary tools
- **Cheat Sheets**: Quick reference guides for game rules and scoring

## Accessing gatool

gatool is available at:
- **Production**: [https://gatool.org](https://gatool.org)
- **Beta**: [https://beta.gatool.org](https://beta.gatool.org)

You will need a login to access the tool's team details editor. See the [Game Announcer Resource Page](https://www.firstinspires.org/resource-library/frc/game-announcer-resources) for details on how to register for an account.

## System Requirements

- **Modern Web Browser**: Chrome, Firefox, Safari, or Edge (latest versions)
- **Internet Connection**: Required for initial setup and data sync (optional after initial load)
- **Recommended Screen Size**: 10" tablet or larger for best experience
- **Authentication**: Google OAuth or Auth0 account

## Getting Started

1. **Navigate to gatool**: Visit [gatool.org](https://gatool.org)
2. **[Optional] Sign In**: Use your authorized Google account or Auth0 credentials
3. **Select Mode**: Choose FRC or FTC mode based on your event
4. **Select Year**: Choose the competition season
5. **Select Event**: Pick your event from the dropdown list
6. **Start Announcing**: Navigate through the tabs to access different features
